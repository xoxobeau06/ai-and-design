✿ words that feel like handwriting ✿ 04 / Typography
§ 03
Type
System
Four typefaces, four registers. Script for the big emotional moments. Handwriting for personal labels. Typewriter for texture and structure. Serif for everything that needs to be read comfortably.
Display — Big Headlines
A cozy morning in the studio
Your name, page titles, the big emotional moments. Used sparingly — when it appears, it means something.
Accent — Personal Moments
a note, a label, a small thought ✿
Labels, captions, ribbon text, small annotations. Feels like your own handwriting on the page.
Typewriter / Mid — Texture
Section Title — Project Label — Caption
Worn, imperfect, textured. Section headers, ribbon labels, captions.
Serif / Body — Readability
The system supports long reading. Rhythm, breath, warmth.
All running text, project descriptions, about page. Never competes with Script above it.
